Suspicious Items - full ranking

The complete list of items, ordered by how suspicious they look. Use this page to triage your manual review work.

How to use this page

1
Use the "Sort by" dropdown to choose how the table is ordered. Default is the composite suspicion score, but you can sort by any single signal (% Iso-Forest, % Revised, etc.).
2
Set "Min rows" to 3 or more if you want to ignore items that only have a couple of observations (their percentages are noisy with so little data).
3
Search for a specific item code using the search box - handy when a planner reports a problem on a particular product.
4
Click "Inspect" on any row to open the per-item diagnostic page (full time-series, anomaly markers, snapshot overlay).
